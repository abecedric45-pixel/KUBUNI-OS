# KUBUNI Browser - Color System

### Primary Green

* Hex : #009B77
* Usage : boutons principaux, IA, actions positives

### Deep Blue

* Hex : #081225
* Usage : arrière-plan principal

### Surface

* Hex : #0F1B33
* Usage : cartes, menus, panneaux

### Secondary Surface

* Hex : #1A2742
* Usage : champs de recherche, boutons secondaires

### Accent Orange

* Hex : #FF7A00
* Usage : notifications, profil, éléments importants

### Gold

* Hex : #D4AF37
* Usage : badges premium, récompenses

### White

* Hex : #FFFFFF

### Light Gray

* Hex : #BFC7D5
