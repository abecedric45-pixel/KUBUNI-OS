# KUBUNI OS

Version : 0.1 Alpha

Mission :
Construire un système d'exploitation mobile africain moderne, sécurisé et souverain.

Objectifs :
- Fournir une alternative Android adaptée à l'Afrique
- Intégrer les services financiers locaux
- Favoriser l'innovation numérique africaine
- Créer un écosystème technologique indépendant

Statut :
En développement.
